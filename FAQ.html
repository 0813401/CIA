關於線上營隊  
Q : 請問線上營隊，會有什麼活動呢？
A : 線上營隊由三大核心活動組成，分別是
1. 精實課程
  • 「國內頂尖教授」分享科系未來趨勢及系所介紹，帶給你「第一手」大學端資訊
  • 「該系專業講師」實際講授基礎課程，讓你提早體驗大學教學內容，減少想像與現實的落差
    • 2. 線上實作
  • 透過實作程式專案、企畫書或電路板操作等該系課程內容，將課程知識學以致用
  • 該系專業講師實際講授基礎課程，讓你提早體驗大學教學內容，減少想像與現實的落差
    • 3. 學長姐分享
  • 「該系學長姊」分享心路歷程與建議，通過參考過來人經歷，減少學生試錯的成本
  • 特別邀請「已畢業」學長姊，以學校所學及職涯出路進行剖析，帶來更全方位的觀點
    • 除了三大核心活動外，團隊另外針對科系特色安排競賽遊戲及團隊計分，增加營隊趣味性的同時，團隊計分讓同學們更專注其中。
    Q : 請問參加線上營隊，對我有什麼幫助呢？
    A : 1. 快速科系探索
    C.I.A線上營隊調查大學生最想在高中就知道的資訊，並經過整理後開發出一系列快速了解科系的課程及活動，讓高中生能在最短的時間內，探索到最真實的科系內容。
    2. 有效釐清目標
    通過學長姐親自分享科系甘苦談，外加進行科系基礎課程實作，以最直接的方式讓你接受第一手資訊，並實際體驗科系所學，讓你提早釐清自己未來目標，降低迷茫產生的焦慮感。
    3. 充實學習歷程
    實際訪問多位教授後，發現學生只要在學習歷程中展現動手實踐的積極度，通常都會是相對加分的，於是C.I.A團隊特地設計線上實作課程，讓同學能不只學到許多知識，還能帶著自己的作品滿載而歸。
    Q : 請問參加線上營隊，該做什麼準備呢？
    A : 你會需要
    1. 有鏡頭的電腦或筆記型電腦
    2. 良好的網路環境
    3. 願意學習新知且享受探索的熱情
    Q : 參加完線上營隊後，想更深入了解科系，該怎麼做？
    A : 我們會做一個比較表(科系探索網站、校系博覽會、實體營隊及個人諮詢)
    Q : 如果有想了解的科系，但沒有相關的線上營隊，該怎麼辦？
    A : 在線上營隊網頁最下方我們設有「許願池」，我們會定時選出大家最想了解的科系辦理營隊，快來許願自己有興趣的營隊吧!
      關於個人諮詢  
    Q : 我能在個人諮詢，得到什麼幫助？
    A : 1. 科系諮詢
    你可以在科系諮詢中找到來自各校系的學長姊，以1對1諮詢的方式解決你心中對大學科系的任何疑惑喔!
    2. 競賽輔導
    你可以在競賽輔導中找到擁有各式競賽經驗的學長姊，像是小論文、程式競賽或是英文檢定，都可以在這裡進行教學喔!
    3. 備審指導
    你可以在備審指導中找到擁有各系申請經驗的學長姊，若對申請學校、書寫備審或準備面試有疑慮都可以來這裡尋求幫助喔!
    Q : 請問如果預約老師後，他沒在約定時間內上線，該怎麼辦？
    A : 請點選主頁聯絡我們告知情形，我們會全額退款並給予課程優惠卷，也會針對該老師的問題進行懲處。
    Q : 請問臨時有事，可以取消預約嗎？
    A : 根據平台規定，於開課24小時前提出請假即可免費取消預約，我們會全額退還給你，而考量到教師權益，若在開課24小時內才提出平台將不予退還費用。
    Q : 請問諮詢上課前，要做什麼準備？
    A : 1. 有鏡頭的電腦或筆記型電腦
    2. 良好的網路環境
    3. 事先跟老師連絡確認需求
    Q : 如何挑到適合自己的老師呢？
    A : 1. 照你的需求選擇科目或科系進行篩選
    2. 觀看教師自我介紹影片及學經歷選擇老師
    3. 參考過去學生反饋及評分進行決定
    4. 經驗豐富且優秀的老師通常費用會較貴，可斟酌選擇
    5. 在課程結束後給予真實回饋，讓平台更公開透明
